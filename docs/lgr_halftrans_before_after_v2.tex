\section*{The TPFA formula}
% ---------------------------------------------------------------

For any cell--face pair the TPFA half-transmissibility is
\begin{equation}
  T_\mathrm{half}
    = k\;\frac{A\;|\mathbf{d}\cdot\hat{\mathbf{n}}|}{|\mathbf{d}|^{2}},
  \label{eq:halftrans}
\end{equation}
where $\mathbf{d}$ is the vector from the cell centroid to the chosen
face point. Decomposing $\mathbf{d} = d_\perp\hat{\mathbf{n}} +
\mathbf{d}_\parallel$ gives
\begin{equation}
  T_\mathrm{half}
    = k\;\frac{A\;d_\perp}{d_\perp^{2} + |\mathbf{d}_\parallel|^{2}}.
\end{equation}
When $\mathbf{d}_\parallel = \mathbf{0}$ this is the K-orthogonal
formula $T_\mathrm{half} = kA/d_\perp$.  Any tangential component
$\mathbf{d}_\parallel \neq \mathbf{0}$ reduces $T_\mathrm{half}$.

The interface transmissibility is the harmonic mean:
\begin{equation}
  T_{CL} =
  \left(\frac{1}{T_{\mathrm{half},C}}+
        \frac{1}{T_{\mathrm{half},L}}\right)^{-1}.
\end{equation}

% ---------------------------------------------------------------
\section*{Before (original code)}
% ---------------------------------------------------------------

\subsection*{How the face centre was computed}

For the coarse cell $C$, the code retrieved the \emph{parent intersection}
at level~0 (the full coarse face $F_C$ before LGR subdivision) via the
function \texttt{getParentIntersectionFromLgrBoundaryFace}, and used its
geometric centre $\mathbf{x}_{F_C}$:

\begin{lstlisting}
const auto& parentIntersection =
    grid_.getParentIntersectionFromLgrBoundaryFace(intersection);

// coarse cell: parent face centre
inside.faceCenter = (intersection.inside().level() == 0)
    ? parentIntersection.geometry().center()         // x_{F_C}
    : grid_.faceCenterEcl(inside.elemIdx, ...);      // x_{F_L}

outside.faceCenter = (intersection.outside().level() == 0)
    ? parentIntersection.geometry().center()         // x_{F_C}
    : grid_.faceCenterEcl(outside.elemIdx, ...);     // x_{F_L}
\end{lstlisting}

\subsection*{Distance vectors and half-transmissibilities}

\begin{align}
  \mathbf{d}_C^{\,\text{before}} &=
    \mathbf{x}_{F_C} - \mathbf{x}_C,
  \label{eq:dC_before}\\[6pt]
  \mathbf{d}_L &=
    \mathbf{x}_{F_L} - \mathbf{x}_L.
  \label{eq:dL}
\end{align}

\begin{align}
  T_{\mathrm{half},C}^{\,\text{before}}
    &= k_C\;\frac{A_L\;\bigl|(\mathbf{x}_{F_C}-\mathbf{x}_C)
                               \cdot\hat{\mathbf{n}}\bigr|}
                 {|\mathbf{x}_{F_C}-\mathbf{x}_C|^{2}},
  \label{eq:TC_before}\\[10pt]
  T_{\mathrm{half},L}^{\,\text{before}}
    &= k_L\;\frac{A_L\;\bigl|(\mathbf{x}_{F_L}-\mathbf{x}_L)
                               \cdot\hat{\mathbf{n}}\bigr|}
                 {|\mathbf{x}_{F_L}-\mathbf{x}_L|^{2}}.
  \label{eq:TL_before}
\end{align}

\subsection*{What went wrong}

On a perfectly Cartesian grid one expects
$\mathbf{x}_{F_C}$ to lie directly in the normal direction from
$\mathbf{x}_C$, giving $\mathbf{d}_\parallel = \mathbf{0}$ and the
correct K-orthogonal result.

However, $\mathbf{x}_{F_C}$ was obtained from
\texttt{getParentIntersectionFromLgrBoundaryFace}, a function that
traverses internal LGR data structures to reconstruct the level-0
intersection from a leaf-level sub-face intersection.
In practice this traversal returned \textbf{inconsistent or slightly
incorrect} face centres for different sub-faces of the same parent face.
The centre $\mathbf{x}_{F_C}$ was therefore not identical for all
sub-faces, breaking the assumption that
$T_{\mathrm{half},C}^{\,\text{before}}$ is the same for every $L_i$
that shares the parent face $F_C$.
The result was \textbf{asymmetric transmissibilities} at the LGR
boundary, observable as an asymmetric pressure field in symmetric test
cases such as \texttt{HORIZONTAL-WELL-REFINED.DATA}.

% ---------------------------------------------------------------
\section*{After (current code)}
% ---------------------------------------------------------------

\subsection*{How the face centre is computed}

\texttt{getParentIntersectionFromLgrBoundaryFace} is no longer called.
Both cells start from the actual sub-face centre obtained via
\texttt{faceCenterEcl}.  For the coarse cell only, the tangential
component of the resulting distance vector is then removed by
projecting onto $\hat{\mathbf{n}}$:

\begin{lstlisting}
// Both cells use the actual sub-face centre
inside.faceCenter  = grid_.faceCenterEcl(inside.elemIdx,  ...); // x_{F_L}
outside.faceCenter = grid_.faceCenterEcl(outside.elemIdx, ...); // x_{F_L}

const auto nhat = intersection.centerUnitOuterNormal();

// For the coarse cell: project to remove tangential component
auto projectCoarseFaceCenter = [&](DimVector& fc, unsigned idx) {
    const auto d    = distanceVector_(fc, idx); // x_{F_L} - x_C
    const Scalar dp = Dune::dot(d, nhat);        // d_perp
    for (unsigned k = 0; k < dimWorld; ++k)
        fc[k] -= d[k] - dp * nhat[k];           // subtract d_parallel
    // result: fc - x_C = dp * nhat  (purely perpendicular)
};

if (intersection.inside().level() == 0)
    projectCoarseFaceCenter(inside.faceCenter,  inside.elemIdx);
else
    projectCoarseFaceCenter(outside.faceCenter, outside.elemIdx);
\end{lstlisting}

\subsection*{Distance vectors and half-transmissibilities}

Let $d_\perp = (\mathbf{x}_{F_L} - \mathbf{x}_C)\cdot\hat{\mathbf{n}}$.
After the projection, the coarse cell's distance vector is:
\begin{align}
  \mathbf{d}_C^{\,\text{after}} &= d_\perp\,\hat{\mathbf{n}},
  \label{eq:dC_after}\\[6pt]
  \mathbf{d}_L &= \mathbf{x}_{F_L} - \mathbf{x}_L
  \quad\text{(unchanged)}.
\end{align}

\begin{align}
  T_{\mathrm{half},C}^{\,\text{after}}
    &= k_C\;\frac{A_L\;d_\perp}{d_\perp^{2}}
     = \frac{k_C\,A_L}{d_\perp}
     = \frac{k_C\,A_L}
            {(\mathbf{x}_{F_L}-\mathbf{x}_C)\cdot\hat{\mathbf{n}}},
  \label{eq:TC_after}\\[10pt]
  T_{\mathrm{half},L}^{\,\text{after}}
    &= k_L\;\frac{A_L\;\bigl|(\mathbf{x}_{F_L}-\mathbf{x}_L)
                               \cdot\hat{\mathbf{n}}\bigr|}
                 {|\mathbf{x}_{F_L}-\mathbf{x}_L|^{2}}.
  \label{eq:TL_after}
\end{align}

Note that $T_{\mathrm{half},L}^{\,\text{after}} =
T_{\mathrm{half},L}^{\,\text{before}}$: the fine cell is unchanged.

\subsection*{Why this is correct and symmetric}

\begin{itemize}
  \item $\mathbf{d}_C^{\,\text{after}} = d_\perp\,\hat{\mathbf{n}}$
        has $\mathbf{d}_\parallel = \mathbf{0}$ by construction.
        The K-orthogonal formula $k_C A_L / d_\perp$ is always recovered.

  \item $d_\perp = (\mathbf{x}_{F_L} - \mathbf{x}_C)\cdot\hat{\mathbf{n}}$
        is the distance from $\mathbf{x}_C$ to the \emph{face plane}.
        Since all sub-faces $F_{L_1}, F_{L_2}, \ldots$ of the same parent
        face $F_C$ lie on the same plane with the same $\hat{\mathbf{n}}$,
        this value is \textbf{identical for every sub-face}.
        $T_{\mathrm{half},C}$ is therefore the same for all $L_i$:
        symmetry is guaranteed by construction, with no dependence on
        internal LGR data structures.

  \item \texttt{getParentIntersectionFromLgrBoundaryFace} is no longer
        called, eliminating the source of the inconsistency.
\end{itemize}
